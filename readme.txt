需要安装 .Net 框架 4.8

不支持 32bit 系统，仅支持 64bit 系统

======================================================================

【常见问题】
Q：是否一定要安装.Net框架4.8？
A：如果能打开程序就不需要安装（现在的 Win10 都自带，Win7 可能需要安装）

Q：哪里下载.Net框架4.8？
A：https://dotnet.microsoft.com/download/dotnet-framework/thank-you/net48-web-installer

Q：修改器无法捕捉进程怎么办？
A：一般是权限不足，无法列举系统进程列表。以管理员方式运行即可。
如果还是无法解决，可以根据程序显示的错误代码到这里查找原因：
https://docs.microsoft.com/en-us/windows/win32/debug/system-error-codes
